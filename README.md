# AI Resume Analyzer

An AI-powered resume analysis tool built with React, TypeScript, Tailwind CSS, and Puter.js. Users can upload resumes, store them securely, and receive AI-generated job-matching feedback and ATS-style scores. The project features modern UI components, seamless authentication, and a clean, reusable code structure.

## Introduction

This application provides an interactive platform for analyzing resumes using AI. Users can authenticate, upload resumes, store them in the cloud, and match them against job listings. The system generates feedback and ATS-style scores using AI capabilities integrated directly into the frontend through Puter.js. The interface is built with modular, reusable components to ensure maintainability and a smooth user experience.

---

## Tech Stack

* **React** – Component-based JavaScript library for building user interfaces.
* **React Router v7** – Client-side routing with nested routes, loaders, and actions.
* **Puter.js** – Client-side SDK enabling authentication, storage, databases, and AI features without a backend.
* **Tailwind CSS** – Utility-first styling for fast UI development.
* **TypeScript** – Strongly-typed JavaScript for reliability and improved tooling.
* **Vite** – Fast development bundler with instant HMR.
* **Zustand** – Lightweight global state management with minimal boilerplate.

---

## Features

* **Browser-based Authentication**
  User login and session handling powered by Puter.js.

* **Resume Uploading & Storage**
  Store multiple resumes securely with full client-side integration.

* **AI Resume Matching**
  Compare resumes against job descriptions and receive ATS scores and feedback.

* **Reusable Component Architecture**
  Clean and modular UI elements for maintainability.

* **Modern UI/UX**
  Responsive design built with Tailwind CSS.

* **Cross-Device Compatibility**
  Works smoothly on desktop, tablet, and mobile devices.

## Quick Start

### Prerequisites

Ensure the following are installed on your machine:

* Git
* Node.js
* npm

### Installation

Clone the project:

```bash
git clone <your-repo-url>
cd <your-project-directory>
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and navigate to:

```
http://localhost:5173
```
